# Spotify Clone

A responsive Spotify-like music player web interface built with HTML and CSS.

## Features
- Music player controls (play/pause, forward/backward)
- Album cards/grid
- Heart icons for likes
- Library section
- Static design mimicking Spotify UI

## How to Run
1. Open `index.html` in any web browser.
2. No build tools or server required – pure HTML/CSS.

## Assets
- Images in `/assets/` folder for player icons, album arts, logo.

Built as a frontend clone project.
